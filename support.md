# WhatBot Support

Need help with WhatBot? This page explains where to get support and what information to include.

## Getting Help

For support, please join the official WhatBot Discord server:

[Join the WhatBot Support Server](https://discord.gg/Dv9CqZTXdV)

You can ask questions, report issues, request help with setup, or share feedback.

## Before Asking for Help

Please check the following first:

- The bot has the correct permissions in your server
- The bot role is high enough in the server role list
- You have used `/setup` if required
- The relevant auction, giveaway, basket, or admin command is being used in the correct channel
- Discord slash commands have finished syncing

## What to Include

When reporting an issue, please include:

- What command or button you used
- What you expected to happen
- What actually happened
- Any error message shown in Discord
- Whether the issue happens for everyone or just one user
- A screenshot if it helps explain the problem

Please do not share private tokens, passwords, payment information, or sensitive server data.

## Bug Reports

If you find a bug, please report it in the support server with as much detail as possible.

Useful details include:

- Server setup
- Command used
- Steps to reproduce the issue
- Approximate time the issue happened
- Any visible error message

## Feature Requests

Feature suggestions are welcome.

Not every suggestion will be added, but feedback helps shape future updates.

## Payments and Transactions

WhatBot helps servers run auctions, giveaways, baskets, and payment-status workflows.

WhatBot does not process payments directly, store payment card details, or act as the seller. Any payment, order, shipping, refund, or dispute is handled between the server owner/seller and the buyer.
